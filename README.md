# vircpt

# About

`virtcpt` can be used to manage libvirt virtual domain checkpoints, as in:

 * create checkpoints
 * delete checkpoints
 * list checkpoints
 * export checkpoints via NBD server backend
 * hot-add attach crated NBD exports to other virtual machines
